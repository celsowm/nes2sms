# TMS9918 Arizona Technical Symposium Draft
by Karl Guttag (Texas Instruments)

## Abstract
Presents the TMS 9918 Video Display Processor (VDP) which provides color graphics and "object oriented graphics" (sprites).

## Background
Designed to interface computing systems to standard televisions, bridging the gap between simple text and high-resolution color graphics.

## Key Concepts
- **Pattern Graphics**: Screen cells defined by names in a pattern generator.
- **Object Oriented Graphics (Sprites)**: Dynamic characters with explicit horizontal and vertical positioning.
- **Prioritized Display Planes**: Allows for 3-D effects.
