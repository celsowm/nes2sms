# MSX I/O Map (SV738 X'Press)
by MAYER (1995)

## Overview
Port summary for MSX systems, including VDP (TMS9918 compatible) and PSG (AY-3-8910).

## VDP Ports (TMS9918/9929/9938)
- **$98**: VRAM Data (Read/Write).
- **$99**: Status (Read) / Address/Register (Write).
- **$9A**: Palette Data (Write).
- **$9B**: Register Data.

## PSG Ports (AY-3-8910)
- **$A0**: Register Select.
- **$A1**: Data Write.
- **$A2**: Data Read.
