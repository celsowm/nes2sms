# Sega Master System and Game Gear Architecture
by Marat Fayzullin (v1.0, 1996)

## General Architecture
- **CPU**: Zilog Z80 at ~3.3MHz.
- **VDP**: Custom video controller based on TMS9918/9928.
- **PSG**: SN76489 (3 tone channels, 1 noise channel).
- **Interrupts**:
    - Maskable (IRQ): Generated by VDP on VBlank ($0038).
    - Non-Maskable (NMI): Generated by PAUSE button ($0066).

## Memory Map
| Range | Function |
| :--- | :--- |
| $0000-$3FFF | 16kB ROM Page 0 |
| $4000-$7FFF | 16kB ROM Page 1 |
| $8000-$BFFF | 16kB ROM Page 2 / Cartridge RAM |
| $C000-$DFFF | 8kB On-board RAM |
| $E000-$FFFF | Mirror of RAM |

## I/O Ports
- **$DC**: Joypad 1 (Read).
- **$DD**: Joypad 2 (Read, SMS only).
- **$7E**: V-Counter (Read) / PSG (Write).
- **$BE**: VDP Data (Read/Write).
- **$BF**: VDP Status (Read) / VDP Control (Write).
- **$3F**: Gun Port / Nationalization (Write, SMS only).

## Nationalization
- SMS uses tricky writes to $3F and reads from $DD.
- GG uses bit 6 of port $00 (1 = US/EU, 0 = JP).
