# Sega SC-3000H Hardware Notes
by Charles MacDonald

## Introduction
The SC-3000H is a personal computer made by Sega, compatible with several 8-bit systems.

### System Variants
- **SG-1000**: No keyboard.
- **SG-1000 II**: No keyboard, supports SK-1100 external keyboard.
- **SC-3000**: Built-in keyboard (rubber keys).
- **SC-3000H**: Built-in keyboard (hard keys).

## Components
- **CPU**: Z80-A compatible (Sharp LH0080A).
- **VDP**: TMS9929A (Video display controller).
- **PSG**: SN76489A (Programmable sound generator).
- **RAM**: 2Kx8 SRAM (Work RAM).

## Memory Map
Defined by the cartridge. MITEC-2 chip provides /CS for $8000-$BFFF and $C000-$FFFF.

## Z80 Port Map
| Range | PPI | VDP | PSG | Description |
| :--- | :--- | :--- | :--- | :--- |
| $00-$1F | Y | Y | Y | PPI+VDP |
| $20-$3F | N | Y | Y | VDP |
| $40-$5F | Y | N | Y | PPI |
| $80-$9F | Y | Y | N | PPI+VDP |
| $A0-$BF | N | Y | N | VDP |
| $C0-$DF | Y | N | N | PPI |

## Keyboard and I/O
Uses a 1-of-10 decoder (74LS145) for keyboard row select and a PPI (uPD9255) for input.
